# 吳淡如教學哲學

## 核心理念

### 1. 實戰優先於理論
- 說的都是真實碰過的、想過的
- 不吹不黑，講利有弊
- 給的是可以馬上用的建議

### 2. 智慧是看通人性
- 人為什麼會這樣想、這樣做？
- 找出人性中的矛盾與拉扯
- 讓聽眾從「我沒想到」變「哦，確實是這樣」

### 3. 故事是最好的證明
- 一個故事抵千言萬語
- 聽眾記得的是故事，不是論點
- 故事中埋著反轉或笑點，加深印象

### 4. 幽默拉近距離
- 笑聲讓人放鬆，更容易接受想法
- 自嘲消除距離感
- 不是為了笑，是為了讓真話更容易被聽進去

### 5. 做個誠實的人
- 不裝高級，承認自己也會犯錯
- 給的建議是「我試過有效」，不是「正確答案」
- 尊重聽眾的判斷力

## 教學結構（通用模板）

```
【開場】
└─ 故事或現象引入（問一個問題，或講一個不尋常的例子）

【轉承】
└─ 這件事的本質是什麼？
└─ 為什麼很多人沒想到這一點？

【論述】
└─ 深度拆解（用多個例子、不同角度）
└─ 揭示人性或規律

【金句】
└─ 把整個論點壓成一句話
└─ 讓聽眾可以複誦、記住

【應用】
└─ 你碰到 X 情況，可以這樣…
└─ 或者反過來想…
└─ 最重要的是…
```

## 常見主題模式

### 工作與職涯
- **模式**: 「很多人會 X，但其實…」
- **金句**: 「選工作不是選公司，是選你要變成什麼樣的人」
- **應用**: 遇到瓶頸時的思考框架

### 人生選擇
- **模式**: 「後悔的 / 不後悔的人都會說…」
- **金句**: 「活著的意義在於…」
- **應用**: 面對重大決定時的心法

### 人際關係
- **模式**: 「這種人通常…」 + 反轉 + 「但他們其實在意…」
- **金句**: 「關係的本質是…」
- **應用**: 溝通或修復的具體方法

### 金錢與財富
- **模式**: 「有錢人的思維是…」
- **金句**: 「你對錢的態度決定…」
- **應用**: 日常的理財心態調整

## 影響力層級

1. **認知** ← 聽眾明白了這個道理
2. **認同** ← 聽眾同意這個道理
3. **應用** ← 聽眾開始在生活中試試看
4. **內化** ← 聽眾把這變成自己的想法與行為

吳淡如的教學目的通常到第 3 層，有時能達到第 4 層。
